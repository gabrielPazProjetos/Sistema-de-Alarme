--- Sistema de Alarme e Projetos Arduino em C
Este repositório contém projetos desenvolvidos em linguagem C para Arduino, ideais para fins educacionais e demonstrações em aula. Os projetos incluem sistemas de alarme, sensores e controle remoto.

--- Projetos incluídos
- alarme_pir.ino
Sistema de alarme com sensor de presença PIR. Ao detectar movimento, acende um LED ou ativa um buzzer por 5 segundos.
- termometro_lm35.ino
Termômetro digital usando o sensor LM35. Lê a temperatura ambiente e exibe no Serial Monitor.
- alerta_botao.ino
Alerta sonoro ativado por botão. Quando pressionado, aciona um buzzer.
- controle_ir_transmissor.ino
Transmissor infravermelho com botões. Envia comandos IR para acender ou apagar dispositivos remotamente.
- controle_ir_receptor.ino
Receptor infravermelho que interpreta sinais IR e acende ou apaga um LED conforme o comando recebido.

--- Componentes utilizados
- Arduino Uno ou compatível
- Sensor PIR HC-SR501
- Sensor LM35
- Buzzer
- LED
- Botões (push button)
- Módulo IR (transmissor e receptor)
- Resistores e jumpers
- Protoboard
